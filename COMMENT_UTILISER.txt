SI CA NE FONCTIONNE PAS:
------------------------

Probleme 1: "start.bat" ne marche pas
Solution: 
  1. Ouvrir terminal (cmd)
  2. Taper: set ANTHROPIC_API_KEY=
  3. Taper: env\Scripts\python.exe demo_server.py 3000

Probleme 2: Erreur 401
Solution: Verifier que .env contient votre vraie cle API

Probleme 3: Port 3000 occupe
Solution: python demo_server.py 8080
