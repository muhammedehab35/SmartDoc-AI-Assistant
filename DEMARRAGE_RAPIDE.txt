ETAPE 1: INSTALLATION (Une seule fois)
----------------------------------------

1. Creer l'environnement virtuel:
   python -m venv env

2. Installer les dependances:
   env\Scripts\pip install -r requirements.txt

3. Configurer la cle API:
   - Copier .env.example vers .env
   - Editer .env et mettre votre cle Anthropic:
     ANTHROPIC_API_KEY=sk-ant-api03-VOTRE_CLE_ICI


ETAPE 2: DEMARRER LE SERVEUR
-----------------------------

Option A - Script automatique (RECOMMANDE):
   start.bat

Option B - Manuel:
   1. Ouvrir un terminal
   2. Executer: set ANTHROPIC_API_KEY=
   3. Executer: env\Scripts\python.exe demo_server.py 3000


ETAPE 3: UTILISER L'APPLICATION
--------------------------------

1. Le serveur demarre sur http://localhost:3000

2. Ouvrir frontend\index.html dans votre navigateur

3. Cliquer sur l'icone parametres (engrenage en haut a droite)

4. Configurer:
   - URL API: http://localhost:3000
   - User ID: user_muhammad_ehab

5. Sauvegarder et commencer a discuter!

TESTS RAPIDES
-------------

Tester la cle API:
   env\Scripts\python.exe test_api_key.py

Test complet:
   set ANTHROPIC_API_KEY= && env\Scripts\python.exe test_langgraph.py


PROBLEMES COURANTS
------------------

Erreur 401 (cle API invalide):
   1. Verifier que .env contient votre vraie cle
   2. Executer: set ANTHROPIC_API_KEY=
   3. Relancer le serveur

Port 3000 occupe:
   python demo_server.py 8080

Module non trouve:
   env\Scripts\pip install -r requirements.txt
